# Air Bnb Clone

## Description
This project is a backend clone of the popular vacation rental platform, Air Bnb. It showcases my learning and implementation of various backend technologies and concepts, including RESTful API design, database management, and user authentication.

## Features
- User authentication (login, signup)
- CRUD operations for listings and reviews
- Dynamic views using EJS templates
- Error handling middleware
- Data validation and sanitization
- Responsive design with CSS

## Technologies Used
- Node.js
- Express.js
- MongoDB (or your chosen database)
- EJS (Embedded JavaScript templating)
- Git & GitHub for version control

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/air-bnb-clone.git
   ```
2. Navigate to the project directory:
   ```bash
   cd air-bnb-clone
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

## Usage
1. Start the server:
   ```bash
   node server.js
   ```
2. Open your browser and go to `http://localhost:3000` to view the application.
